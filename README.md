# Sport Corner — E-Commerce Website

A simple sports shop website with WhatsApp ordering and an admin dashboard. No backend needed.

## Files

```
sport-corner/
├── index.html   # Client shop
├── admin.html   # Admin dashboard
├── style.css    # Styles
├── script.js    # Logic
└── logo.JPG     # Logo
```


## WhatsApp Number

`+213 540 543 342` — change it in `script.js`:
```js
return `https://wa.me/213540543342?text=${msg}`;
```

## Deploy

Drag the folder to [netlify.com](https://netlify.com) — free & instant.
